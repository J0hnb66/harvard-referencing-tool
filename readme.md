# Harvard Referencing Tool (Flask + JavaScript)

A clean, fast, internally hosted Harvard referencing generator built with Flask and a custom Windows‑11‑style UI.  
Designed for accuracy, speed, and a native‑app feel — without external dependencies or generic web‑app branding.

---

## 🚀 Features

- Live reference preview (Harvard style)
- Supports **Books**, **Journal Articles**, and **Websites**
- Automatic formatting (italics, punctuation, ordering)
- One‑click **Copy** (rich text with italics preserved)
- **Add to List** for building a full reference list
- **History panel** with restore + delete
- **Formatted download** (HTML document, Word‑friendly)
- Fully client‑side UI with Flask backend formatting
- No external branding, no third‑party services

---

## 🛠️ Tech Stack

- **Python 3.x**
- **Flask**
- **HTML / CSS / JavaScript**
- Windows‑11‑style Fluent UI (custom)
- LocalStorage for history persistence
